Fullstack Logging Tool

Developed a full-stack web application using ASP.NET Core and Vue.js frameworks to parse, store, and display log files sent from ERCOT, used REST API Connections to fetch and display data on an internal network webpage. Included functionality for formatted .csv files with parsed ERCOT data and frontend integration with Vuetify 3 for extra GUI features.

Used ASP.NET Core 6.0 to develp a C# backend. The backend was responsible for finding data log files stored in a specified folder, parsed all the data within the log files, and stored these files in a data structure (C# Dictionary) within the backend itself. Designed API Controllers to help facillitate Http[Get], Http[Post], and Http[Get["id"], API calls.

Vue.js was used as a Javascript Framework for the frontend webpage. The Vue.js project was responsible for handling Http[Get], Http[Post], and Http[Get["id"] API calls on the frontend side. Displaying data for Http[Get] and sending data to the C# backend when an Http[Post] is called. The frontend uses a package on the top of the Vue.js framework called Vuetify 3 to create Google Material Design Style Components.

Overall, developed a CRUD application to store and display ERCOT logs using ASP.NET Core 6.0, Vue.js, and RESTful API connections.

Simulated Y-86 CPU

Worked in a team to create a CPU using Logisim Evolution to perform custom-made Y-86 assembly instructions including bubble sort.

Individually Desgined the Register and Fetch Components. The register component is responsible for temporary storage of values during a CPU's execution. The fetch component was responsible for reciving an instruction code as an input, and seperating the instruction code into chunks of bytes the CPU understands how to use.

Collaborated with teammates to assemble our Main CPU together, combining all individual components (fetch, decode, execute, write back, etc). After all components were assembled, our group developed Y-86 assembly code that was functional and able to run small processes including bubble sort.
